# AlphaVerse Design Rules

1. Never redesign existing sections without instruction.
2. Keep the design premium, minimal and Apple-inspired.
3. Use black, white and subtle blue as the primary color palette.
4. Prefer reusable React components.
5. Use TypeScript everywhere.
6. Follow responsive-first design.
7. Keep animations subtle and meaningful.
8. Never use random placeholder content unless requested.
9. Every new feature should be modular.
10. Preserve consistency across all pages.
